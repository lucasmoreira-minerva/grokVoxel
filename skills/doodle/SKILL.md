# Doodle Board skill

Same stage pipeline as GrokVoxel (`beats.json` → keyframes → i2v → TTS → music → assemble),
but the LOOK layer is **marker doodle / whiteboard**, not paper collage.

## Look floor
- Cream or light-gray board background
- Black/blue marker strokes, highlighter accents
- Sticky notes, arrows, underlines
- Flat 2D, not photoreal

## Flow
Follow the same gates and scripts pattern as `skills/grok-voxel/SKILL.md`.
Reuse assemble scripts from grok-voxel until this pack has its own copies.
