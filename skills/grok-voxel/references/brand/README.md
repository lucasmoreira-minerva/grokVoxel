# Brand assets (project-specific)

for use as image-to-image references and on-screen branding in GrokVoxel pilots.

| File | Use |
|---|---|

| `garca-valley-mark-hero.png` | Larger mark for keyframes / thumbnails |
| `garca-valley-lockup.png` | Header lockup (mark + wordmark) |
| `garca-valley-masterplan-plan.png` | Zoning / site plan crop |
| `garca-valley-masterplan-zones.jpeg` | Embedded plan from PDF |
| `garca-valley-satellite-map.png` | Location aerial (large; optional in git) |

Source document: *Garça Valley Masterplan v4.0* (Prefeitura de Garça / Instituto Minerva).
Keep confidential docs out of public forks if required by the client.
